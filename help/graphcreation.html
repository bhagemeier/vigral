How to use this software

graph creation

  • create a vertex
  • create an undirected edge
  • create a directed edge
  • delete a vertex or an edge
  • change parameters
  • select a vertex
  • select multiple vertices
  • move vertices

create a vertex

left-click in a free space of the drawing area




create an undirected edge

drag from a vertex to another by holding the key 'KEY_UNDIRECTED_EDGE' (default: 'u')




create a directed edge

drag from a vertex to another by holding the key 'KEY_DIRECTED_EDGE' (default: 'd')




delete a vertex or an edge

open the context menu of the element you want to delete by right-clicking it and choose the last entry




change parameters

open the context menu of the element you want to edit by right-clicking it and choose the second last entry




select a vertex

simply left-click on the vertex you want to select




select multiple vertices

you have 2 possibilities:

  1. left-click the vertices you want to select by holding the key 'KEY_MULTIPLE_SELECT' (default: 'CTRL')
  2. draw a rect by start a drag in a free space of the drawing area by holding the key 'KEY_RECTANGULAR_SELECT' (default: 'SHIFT'), that selects all vertices within the rectangle when releasing the mouse-button



move vertices

dragging one of the selected vertices will move all selected ones
