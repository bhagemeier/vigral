How to use this software

edit settings

  • edit color settings
  • edit key settings
  • edit view settings
  • restore defaults

edit color settings

  1. choose 'File' -> 'Settings' from the menubar
  2. hit the 'Colors' tab
  3. double-click the center column and change the value
  4. apply the value by hitting 'Return' or click another table cell
  5. apply or decline your changes



edit key settings

  1. choose 'File' -> 'Settings' from the menubar
  2. hit the 'Keys' tab
  3. double-click a row and wait for a dialog to appear
  4. press the desired key
  5. apply or decline this key
  6. apply or decline your changes



edit view settings

  1. choose 'File' -> 'Settings' from the menubar
  2. hit the 'View' tab
  3. check the information that should be displayed
  4. apply or decline your changes



restore defaults

  1. choose 'File' -> 'Settings' from the menubar
  2. hit the 'Colors' or 'Keys' tab
  3. click 'Restore Defaults'
  4. apply or decline your changes
