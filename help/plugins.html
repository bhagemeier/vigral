How to use this software

plugins

  • add a new algorithm to the list
  • remove an algorithm from the list
  • reload algorithms

add a new algorithm to the list

  1. add the '.class' file to the 'plugins' folder in the execution folder
  2. reload plugins



remove an algorithm from the list

  1. remove the '.class' file from the 'plugins' folder in the execution folder
  2. reload plugins



reload algorithms

you have 2 possibilities:

  • restart the application
  • choose 'PlugIn' -> 'Reload' from the menubar
